**Isotope ($ISOP)**

**Your Browser is Now a Mine. No Hardware, Just Code.**

**Abstract:**
Isotope ($ISOP) is a browser-first crypto mining ecosystem built on a Layer 1 BlockDAG architecture using lightweight Proof-of-Work (PoW) nodes. It is designed to make blockchain participation simple, accessible, and energy-efficient. Unlike traditional mining, Isotope leverages idle device resources through browser extensions or progressive web apps (PWA) to generate mining points, allowing everyday users to contribute to the network without specialized hardware.

Users can accumulate points and convert them into $ISOP tokens, participating in ecosystem growth while enjoying a low-barrier entry to Web3. The $ISOP token powers ecosystem utilities including rewards, premium features, staking opportunities, and manual/automatic burn mechanics for a deflationary supply model.

The total token supply is 1,000,000,000 $ISOP, distributed as follows: 50% locked, 45% presale, 2% airdrop, and 3% allocated for development and team operations. This allocation ensures a balanced approach between community growth, network security, and long-term project sustainability.

By combining PoW-based BlockDAG nodes with a lightweight, mobile-ready, and user-consent-driven mining model, Isotope establishes a fair, scalable, and energy-conscious entry point for the next generation of Web3 participants, making any browser a contributing node in the network.
